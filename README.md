Наша программа будет имитировать эволюцию. 
Основная идея взята из книги Ричарда Докинза «Слепой часовщик».
На каждом шаге эволюционного процесса мы будем порождать N созданий (биоморф), представляющих собой геометрические фигуры, составленные из смежных отрезков и обладающие билатеральной симметрией.
Основой для генерации биоморфы является ее генотип, представляющий собой последовательность из 16 генов.
Первые пятнадцать генов (значения от -9 до +9) определяют скелет биоморфы, кодируя положение каждого отрезка. Для генерации используются случайно выбранные 7 генов из 15.
Последний ген (значения от 2 до 12) определяет длину биоморфы.
Путем последовательных мутаций мы будем изменять генотип. В каждом поколении мутации подвергается только один ген, причем изменяться он может только на единицу.
Для каждого поколения необходимо сгенерировать N биоморф, представляющих собой черно-белые изображения размером 150 на 150 пикселей.
Критерием естественного отбора будет являться степень подобия биоморфы заданному результату (например, котику)
Подобие может определяться как эвклидово или манхеттенское расстояние либо как нормализованная кросс-корреляция.
Эволюция начинается со случайно сгенерированного генотипа.
Критерием прекращения отбора будет являться отсутствие увеличения степени подобия на протяжении 10 поколений.
